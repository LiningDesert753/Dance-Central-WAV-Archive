# Dance Central WAV Archive
An archive of every Dance Central song, in WAV format (Work in Progress)

## ABOUT:

This archive aims to not only have the series' songlist, but also any stems that may be present in certain songs, and an archive of the UI music as well.
<br>Each song includes the following variants:
- Main Song
- Break it Down (BiD) stems* **
- Instrumental & Vocal stems*
- Variants of existing songs
	- (EX: Body Movin' (Fatboy Slim Remix) has a different edit in the E3 2010 build than in the final version.)
<br><br>*only if applicable
<br>**the BiD stems are excluded from the main file.
